### 快速开始
###### 运行环境
- Java-21
###### **项目安装**
1. 自行下载编译
下载项目源码
```shell
git clone https://github.com/MuskStark/HMoneta-Agent.git
```
2. 使用Maven在项目跟目录执行编译
3. 使用下面指令运行编译产物
```shell
java -jar ${project.artifactId}-${project.version}.jar
```
